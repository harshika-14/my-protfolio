# 🌌 Aetheria - Futuristic Personal Portfolio

A premium, futuristic, and highly interactive personal portfolio website built with **React 19**, **Vite**, and **Framer Motion**. This project showcases a creative identity through a high-end digital experience, featuring deep visual layers, immersive animations, and interactive labs.

![Portfolio Preview](https://picsum.photos/seed/portfolio-preview/1200/600)

## ✨ Key Features

- **🚀 Immersive UI/UX**: Dark theme with layered radial glows, glassmorphism, and a subtle animated grain overlay.
- **🎨 Live Theme Customizer**: Instantly switch between multiple accent colors (Purple, Blue, Cyan, Pink, Amber) across the entire UI.
- **👕 Interactive T-Shirt Designer**: A mini-lab where users can customize brand merchandise in real-time.
- **🖱️ Mouse-Reactive Spotlight**: A dynamic glow effect that follows the cursor, creating depth and focus.
- **📱 Premium Project Showcase**: 3D tilt cards with smooth fullscreen modals for detailed project exploration.
- **⚡ Technical Arsenal**: Animated progress bars and floating background elements that respond to scroll.
- **💼 Professional Timeline**: An animated vertical journey showcasing career growth and leadership roles.
- **📬 Sexy Contact Section**: Modern form with focus-glow inputs, floating labels, and success animations.
- **🧠 Micro-Interactions**: Custom magnetic cursor, smooth scroll progress, and staggered entry animations.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion (Motion/React)](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **Utilities**: `clsx`, `tailwind-merge`, `react-parallax-tilt`

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/harshika-14/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
